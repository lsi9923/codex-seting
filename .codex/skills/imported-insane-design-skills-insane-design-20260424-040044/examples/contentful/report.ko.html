Contentful — Royal blue + Earth-tone illustration DS

CSS 변수 0개, 하드코딩 hex만 사용하는 마케팅 사이트. Royal blue #1770E6가 CTA를 독점하고, 테라코타/세이지/카키 earth-tone palette가 일러스트레이션을 구성한다.

Step 1 — 폰트
/* 1. 폰트 — Avenir Next + system fallback */
body {
  font-family: "Avenir Next",
    "Segoe UI", -apple-system,
    BlinkMacSystemFont,
    "Helvetica Neue", sans-serif;
  font-weight: 400;
}
Step 2 — 라이트 배경
/* 2. 라이트 배경 + dark text */
:root {
  --bg:      #FFFFFF;
  --bg-muted: #EFF2F6;
  --fg:       #2B2D31;
}
body {
  background: var(--bg);
  color: var(--fg);
}
Step 3 — 브랜드 블루
/* 3. 브랜드 royal blue */
:root {
  --brand:      #1770E6;
  --brand-navy: #2C407D;
}
/* CTA, link, logo → brand only */
/* earth-tone → illustrations */
절대 하지 말 것: Contentful 브랜드는 royal blue #1770E6(HSL 215°)다. cyan-계열 #0286C3는 실제 CSS에 0회 등장 — 잘못 쓰면 즉시 "Contentful 아님"처럼 보인다. earth-tone accent palette(#A86451, #587D69, #8A724D)는 오직 일러스트레이션 전용이며 UI 브랜드 컬러가 아니다.

09 — Rules

자주 틀리는 것들

DO
  • Royal blue #1770E6(HSL 215°)을 브랜드 컬러로 사용. CTA, link, logo 모두.
  • Navy #2C407D / #2E3F70을 darker variant로 사용 (footer, 헤더).
  • Earth-tone accent palette을 일러스트레이션/히어로 이미지에 사용: terracotta #A86451, sage #587D69, khaki #8A724D.
  • Dark text는 #2B2D31 (pure gray, blue-tint 없음).
  • Surface는 #FFFFFF + #EFF2F6 + #DDE5EC 3-step.
  • Avenir Next 폰트 + system fallback (Segoe UI, -apple-system).
  • Radius는 4px(button) / 8px(card) 2종만.
  • Simple single-layer shadow (0 5px 10px rgba(0,0,0,.15)).
DON'T
  • Cyan-계열 블루 (#0286C3, #00AFCC 등) — 실제 CSS에 0회.
  • Dark text를 #1B1E28 같은 blue-tinted로 — 실제는 순수 gray #2B2D31.
  • Avenir Next를 Inter/Roboto로 대체 — geometric 인상 다름.
  • Earth-tone을 UI 브랜드 컬러로 사용 — 오직 일러스트레이션/아이콘 전용.
  • Multi-layer dual shadow (Stripe/Clerk 계열) — Contentful은 single-layer.
  • CSS variable 기반 토큰 시스템 기대 — 마케팅 홈페이지는 하드코딩 hex만.
  • --f36-* Forma 36 토큰을 마케팅 페이지에서 사용 — 앱 UI 전용.
  • 6-step+ radius 시스템 — Contentful은 2-step만 사용.
