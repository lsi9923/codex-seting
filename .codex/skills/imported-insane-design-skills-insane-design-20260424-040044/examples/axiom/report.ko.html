Design System Report

Axiom — Tailwind 유틸리티 퍼스트

https://axiom.co 가 실제로 서빙하는 CSS에서 직접 뽑은 디자인 토큰·타이포그래피·컴포넌트 스펙. BerkeleyMono 모노스페이스 브랜드, 다크 퍼스트, 오렌지 액센트. Tailwind CSS v3 기반.

Quick Start 5분 안에 Axiom처럼 만들기 — 3가지만 하면 80%
1 · 폰트 — monospace brand body { font-family: "BerkeleyMono", ui-monospace, SFMono-Regular, Menlo, Monaco, Consolas, "Courier New", monospace; font-weight: 400; }
2 · 다크 배경 + 텍스트 :root { --bg: #111827; /* gray-900 */ --fg: #FFFFFF; --muted: #9CA3AF; } body { background: var(--bg); color: var(--fg); }
3 · 오렌지 액센트 CTA :root { --accent: #DA5C2C; /* hero CTA / chart lines */ } .btn-primary { background: var(--accent); color: #fff; border-radius: 4px; }
절대 하지 말아야 할 것 하나: Inter 같은 sans-serif로 본문을 쓰면 옵저버빌리티 도구 특유의 "터미널 / SRE 감성"이 완전히 사라진다. Axiom의 브랜드 핵심은 monospace 타이포그래피 (BerkeleyMono)다. 브랜드 CTA도 보라가 아니라 오렌지 #DA5C2C.

03 · Typography

타이포그래피 — monospace가 본문이다

BerkeleyMono를 본문 폰트로 사용하는 것이 Axiom의 핵심 브랜드 차별점. 전체 스케일이 monospace라 equivalent size에서 sans-serif보다 x-height가 낮고 폭이 일정하다.

05 · Spacing

스페이싱(간격 체계) — Tailwind 유틸 직접 사용

Axiom은 spacing을 유틸 클래스로 전부 처리하고 토큰 변수로 노출하지 않는다. Tailwind 기본 spacing 스케일 사용.

구조적 특징: --tw-space-* 커스텀 프로퍼티는 5개뿐 (내부 바인딩용). 스페이싱 API는 전부 유틸 클래스 (.p-4, .gap-6).

07 · Shadows

섀도우 없는 flat 디자인

Axiom은 elevation 없이 border만으로 레이어를 구분한다. 의도적인 terminal 감성 선택.

기본값
대부분의 컴포넌트
box-shadow: none
kbd shadow
키 캡 표시 전용
0 0 0 1px rgb(17 24 39/10%), 0 3px 0 rgb(17 24 39/10%)
border-only elevation
레이어 구분 방식
border: 1px solid #262626
패턴: Axiom은 shadow-heavy 카드를 의도적으로 피함. 레이어는 border: 1px solid #262626으로만 구분. --tw-ring-*은 focus state 전용.

10 · Do / Don't

자주 틀리는 것들

지켜야 할 것과 피해야 할 것.

Do

  • BerkeleyMono를 본문 폰트로 사용 — Axiom의 브랜드 핵심 차별 요소.
  • 다크 배경 #111827 (gray-900) + 카드 #171717 (neutral-900) 투톤 구분.
  • 오렌지 #DA5C2C를 hero CTA와 차트 데이터 포인트에 액센트로 사용.
  • 코드 블록은 #171717 bg + VS Code 계열 syntax colors.
  • shadow-less flat 디자인 — 레이어는 border로만 구분.
  • radius는 4px 일관 사용.
  • severity badge는 Radix 컬러 계열 (#E5484D, #29A383, #0090FF).

Don't

  • Inter나 sans-serif 본문 — Axiom은 monospace 브랜드. sans로 바꾸면 "또 하나의 YC 스타트업 랜딩"처럼 보인다.
  • 보라색 브랜드 — axiom.co CSS에 0회. 오렌지 #DA5C2C가 유일한 브랜드 accent.
  • 화려한 그라디언트 / glow — Axiom은 의도적으로 flat.
  • semantic 토큰 변수 기대 — Axiom 마케팅은 순수 Tailwind 유틸만 사용.
  • 라이트 테마 기본값 — Axiom은 다크 퍼스트 (#111827이 bg).
  • sans-serif와 monospace 혼용 — 전체가 mono여야 Axiom 특유의 질감이 나온다.
  • Shadow-heavy 카드 — Axiom은 shadow 거의 0 (kbd 제외).
