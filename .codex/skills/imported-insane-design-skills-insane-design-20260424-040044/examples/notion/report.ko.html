Notion — Warm Ink + 1,111 토큰

Next.js 기반 자체 3-layer token indirection 시스템. pure black이 아닌 warm ink #37352F의 알파 5단계가 정체성이다. 3-way 폰트 토글(NotionInter / Lyon Text / iA Writer Mono)은 블록별 동적 전환.

Step 1 — 3-Way 폰트
body {
  font-family: "NotionInter",
    ui-sans-serif, -apple-system,
    BlinkMacSystemFont, sans-serif;
}
.serif { font-family:
  "Lyon Text", Georgia, serif;
}
code, pre { font-family:
  "iA Writer Mono", "Nitti",
  Menlo, monospace;
}
Step 2 — Warm Ink
:root {
  --color-ink-primary: #37352F;
  /* warm black, not #000 */
  --color-ink-muted:   #37352FA6;
  /* α65 */
  --color-ink-faint:   #37352F99;
  --color-ink-divider: #37352F29;
  --color-bg-page:     #FFFFFF;
  --color-bg-soft:     #F7F7F5;
  --color-border:      #E9E9E7;
}
body {
  background: var(--color-bg-page);
  color: var(--color-ink-primary);
}
Step 3 — 링크 블루
:root {
  --color-link:     #2383E2;
  --color-link-alt: #097FE8;
}

/* Campaign (일시적) */
:root {
  --color-campaign-navy:
    #2537B1; /* AI Agents 캠페인 */
}

/* Block callout 9색 팔레트 */
.callout--blue {
  background: #E7F3F8;
  color: #337EA9;
  border-radius: 3px;
}
절대 하지 말 것: 본문 text 색을 pure black #000000으로 쓰지 말 것. Notion의 진짜 정체성은 warm ink #37352F(갈색 섞인 따뜻한 검정)와 그 알파 변종 5단계 (CC/A6/99/29/17)다. 순흰색 배경에 순검정 텍스트로 만들면 Notion의 "paper-like" 느낌이 사라진다.

스페이싱(간격 체계) — 8pt 그리드가 아닌 20 단위

304개 spacing 변수. 일반적인 8pt 그리드 아님. 20/40/60 단위가 주력이며 블록 내부는 20/24/32 혼용.

--spacing-s / --spacing-m 40px small (의도적으로 s=m)
--marquee-padding-y 50px marquee vertical

Rules

자주 틀리는 것들

pure black 텍스트와 Inter 단일 폰트 사용이 가장 흔한 실수. 캠페인 색을 브랜드로 착각하는 것도 주의.

DO
  • 본문 text는 warm ink #37352F. 알파 5단계 (CC/A6/99/29/17)로 구분.
  • Warm off-white #F7F7F5를 sidebar/panel 배경으로.
  • 폰트는 NotionInter + Lyon Text + iA Writer Mono 3-way 토글.
  • 13단계 font-size 시스템 (font-size-50 ~ font-size-1000) 그대로 사용.
  • Letter-spacing optical compensation — 사이즈별 + weight별 미세 보정.
  • 링크는 #2383E2. #2EAADC 아님.
  • Shadow는 3-level composite. shadow-level-200은 4-layer stacking.
  • Callout/tag는 9색 팔레트에서 text/bg pair로.
DON'T
  • Pure black #000 text 금지 → 반드시 warm ink #37352F.
  • Inter 단일 폰트 금지 → NotionInter + Lyon Text (serif) + iA Writer Mono (mono).
  • #2EAADC 링크 블루 금지 → 실제는 #2383E2.
  • #F7F6F3 cool gray 금지 → warm #F7F7F5.
  • 8pt 그리드 가정 금지 → Notion은 20/24/32/40/60 혼합.
  • "Notion is flat, no shadows" 가정 금지 → 실제는 4-layer composite micro-shadow.
  • 6단계 type scale 금지 → 13단계 필수.
  • Campaign 색 (#2537B1 navy 등)을 영구 브랜드로 착각 금지.
