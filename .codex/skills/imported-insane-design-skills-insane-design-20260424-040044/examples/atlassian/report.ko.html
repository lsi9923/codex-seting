Design System Report

Atlassian — --ds-*

https://www.atlassian.com 이 실제로 서빙하는 CSS에서 직접 뽑은 디자인 토큰·타이포그래피·컴포넌트 스펙. ADS v2 기반, 490개 커스텀 프로퍼티를 직접 파싱한 결과이며 AI 추론이나 추측은 섞이지 않았다.

Quick Start 5분 안에 Atlassian처럼 만들기 — 3가지만 하면 80%
1 · 폰트 + weight 400 body { font-family: "Atlassian Sans", ui-sans-serif, -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif; font-weight: 400; }
2 · 배경 + 텍스트 :root { --bg: #FFFFFF; --fg: #292A2E; } body { background: var(--bg); color: var(--fg); }
3 · 브랜드 블루 + 헤딩 :root { --brand: #1868DB; } h1,h2,h3,h4,h5,h6 { font-weight: 653; /* Atlassian Sans 가변폰트 비표준 bold weight */ }
절대 하지 말아야 할 것 하나: 구 브랜드 블루 #0052CC를 쓰지 말 것. 현재 ADS v2의 브랜드는 #1868DB이며 CSS 빈도 596회로 절대 다수를 차지한다. 또한 헤딩 볼드는 700이 아니라 653 — Atlassian Sans 가변 폰트의 비표준 weight다.

타이포그래피 — 헤딩 볼드는 653이다

05 · Spacing

스페이싱(간격 체계) — rem 기반 14단

rem 기반 14-step 스케일. 토큰 번호 × 0.125rem = value (100 = 0.5rem = 8px).

네이밍 규칙: 토큰 번호 × 0.125rem = value. 즉 --ds-space-200 = 1rem = 16px. Negative scale도 동일한 step으로 존재 (--ds-space-negative-025 ~ -400). 마진-오버랩/풀블리드 컴포넌트용.

07 · Shadows

섀도우는 항상 두 겹

모든 elevation이 dual-shadow 원자 (offset + hairline outline). 색상은 모두 #1E1F21 + 알파 차등.

--ds-shadow-raised
카드 리프트 (2-layer)
0px 1px 1px #1E1F2140, 0px 0px 1px #1E1F214F
--ds-shadow-overlay
popover / dropdown
0px 8px 12px #1E1F2126, 0px 0px 1px #1E1F214F
--ds-shadow-overflow
스크롤 끝단 페이드
0px 0px 8px #1E1F2129, 0px 0px 1px #1E1F211F
패턴: 모든 elevation이 dual-shadow 원자 — offset shadow + hairline outline 1px을 겹치는 구조. 단층 box-shadow로 치환하면 깊이감이 무너진다. 색상은 모두 #1E1F21 기반 알파 차등.

10 · Do / Don't

자주 틀리는 것들

지켜야 할 것과 피해야 할 것.

Do

  • #1868DB를 브랜드 블루로 사용 — --ds-text-brand, --ds-link, --ds-background-brand-bold 모두 이 값.
  • 헤딩에 font-weight: 653 적용 — Atlassian Sans 가변 폰트 전용 비표준 weight.
  • 마케팅 사이트엔 Charlie Display + Charlie Text, 앱 UI엔 Atlassian Sans 분리.
  • 10개 accent family를 subtlest + bolder 페어로 lozenge/badge에 사용.
  • rem 기반 spacing 리듬 유지 — 토큰 번호 × 0.125rem = value.
  • dual-shadow 원자 (0 1px 1px + 0 0 1px)로 card elevation 표현.
  • border 색상은 알파-블랙 (#0B120E24, 14%) 사용 — 배경에 자연스럽게 적응.
  • 다크 모드는 data-color-mode="dark" 어트리뷰트로 스위칭.

Don't

  • #0052CC 구 브랜드 블루 사용 — legacy alias 28회만 남음. 현 브랜드는 #1868DB 596회.
  • font-weight: 700 사용 — Atlassian Sans의 bold는 653이다.
  • color-brand/color-gray-* 같은 flat 네이밍 기대 — 실제는 --ds-text-brand, --ds-background-neutral-* 시맨틱.
  • 마케팅 헤딩에 Atlassian Sans 강제 — Charlie Display가 브랜드 자산.
  • 단일 shadow 사용 — Atlaskit은 모든 elevation을 dual-layer로 구성.
  • radius 토큰 변수 (--radius-md) 기대 — Atlaskit은 컴포넌트에 하드코딩.
  • purple(#803FA5)을 보조 브랜드로 취급 — discovery/new 상태 전용 semantic color다.
