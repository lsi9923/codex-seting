Design System Report

Cal.com — Framer + --framer-*

https://cal.com 마케팅 사이트가 실제로 서빙하는 CSS에서 직접 뽑은 디자인 토큰·타이포그래피·컴포넌트 스펙. Framer 호스티드 사이트. Cal Sans display font + Inter body. link state 2색 페어가 브랜드다.

Quick Start 5분 안에 Cal.com처럼 만들기 — 3가지만 하면 80%
1 · 폰트 분리 :root { --font-display: "Cal Sans", "Inter Display", Inter, system-ui, sans-serif; --font-body: "Inter", "Inter Variable", system-ui, sans-serif; } body { font-family: var(--font-body); } h1,h2,h3 { font-family: var(--font-display); }
2 · 라이트 배경 + 텍스트 :root { --bg: #FFFFFF; --fg: #141414; /* heading용 */ --fg-display: #242424; } body { background: var(--bg); color: var(--fg); }
3 · link state 2색 페어 :root { /* link default */ --link: #0099FF; /* active / current */ --link-current: #6349EA; } a { color: var(--link); } a[aria-current="page"] { color: var(--link-current); }
절대 하지 말아야 할 것 하나: Cal.com은 오렌지 브랜드가 아니다. 실제 accent는 #6349EA 보라와 #0099FF 파랑 2색 페어. 오렌지 #FF7A45는 CSS에 0회. 헤드라인에 반드시 Cal Sans 커스텀 폰트 — 이것이 Cal.com 브랜드의 핵심.

03 · Typography

타이포그래피 — Cal Sans가 브랜드다

Display는 Cal Sans (오픈소스 — GitHub: calcom/font), body는 Inter. Cal Sans의 semibold가 Inter Display의 semibold보다 시각적으로 훨씬 distinctive.

05 · Spacing

스페이싱(간격 체계) — Framer 레이어 numeric

Framer는 spacing을 개별 layer의 numeric value로 관리. --space-md 같은 공용 scale 토큰 개념이 없다.

Framer 구조적 특징: --framer-font-size: 16px, --framer-letter-spacing: 0만 토큰화. 나머지는 레이어 단위 numeric. 재현할 때 권장 heuristic: small gap 8px, medium 16px, section padding 64~80px, hero rhythm 120px.

  --font-mono:    "Roboto Mono", ui-monospace, Menlo, monospace;

  --bg:         #FFFFFF;
  --bg-section: #F4F4F4;

  /* Borders */
  --border:     #E1E2E3;

  /* Accents (link state 3) */
  --link:         #0099FF;
  --link-current: #6349EA; /* brand purple */

  /* Radius */
  --radius-card:  16px;
  --radius-btn:   12px;
  --radius-pill:  9999px;
}

body {
  font-family: var(--font-body);
  font-weight: 400;
  background: var(--bg);
  color: var(--text);
}

09 · Do / Don't

자주 틀리는 것들

지켜야 할 것과 피해야 할 것.

Do

  • Cal Sans 폰트를 display/heading에 사용 — Cal.com 브랜드의 핵심 차별 요소 (오픈소스, GitHub calcom/font).
  • Body는 Inter / Inter Variable.
  • 텍스트 컬러는 #141414 (body) + #242424 (heading) 페어.
  • 링크는 2색 state 페어: 기본 #0099FF 파랑, current #6349EA 보라.
  • 라이트 테마 기본값 — 페이지 bg #FFFFFF, 섹션 bg #F4F4F4.
  • 카드 radius 16px 일관. 버튼은 pill (9999px) 또는 12px.
  • Flat 디자인 — border-only elevation, shadow 최소.

Don't

  • 오렌지 CTA 사용 — Cal.com CSS에 오렌지 #FF7A45는 0회. 로고 장식 색을 브랜드로 착각한 것.
  • 헤딩을 Inter로만 처리 — Cal Sans 누락 시 브랜드 정체성 소실.
  • 단일 브랜드 컬러 기대 — Cal.com은 link state 페어가 브랜드.
  • Tailwind spacing scale 기대 — Framer는 개별 layer numeric value로 관리.
  • 단일 shadow 시스템 — Cal.com은 shadow 거의 없는 flat design.
  • Cal Sans를 body로 쓰기 — semibold만 있어서 long-form 가독성 떨어짐.
잘못된 값 올바른 값 이유
Orange CTA #141414 (dark) Cal.com CTA는 오렌지가 아닌 검정 pill 버튼.
Inter (heading) Cal Sans (heading) Cal Sans가 브랜드 핵심. Inter는 body용.
single brand color #0099FF + #6349EA 페어 Cal.com은 link state 2색 페어가 accent 전략.
