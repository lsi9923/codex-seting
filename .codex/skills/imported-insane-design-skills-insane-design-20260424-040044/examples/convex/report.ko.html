Convex — Cream background + Monokai art DS

크림/아이보리 warm backgrounds 위에 Monokai syntax palette가 얹힌 art/magazine 감성. GT America + Publico(serif) + VCR(pixel) 3-폰트 조합과 Crayola crayon 토큰 네이밍이 브랜드 아이덴티티.

Step 1 — 폰트 4종
/* 1. GT America + Publico + VCR */
:root {
  --font-sans:  "GT America", ui-sans-serif,
    system-ui, -apple-system, sans-serif;
  --font-serif: "Publico", Georgia, serif;
  --font-pixel: "VCR", "VT323", monospace;
  --font-mono:  ui-monospace, Menlo, monospace;
}
body { font-family: var(--font-sans); }
Step 2 — Cream 배경
/* 2. 크림/아이보리 배경 + 코코아 텍스트 */
:root {
  --bg-milk:     #FFFDF4; /* primary bg */
  --bg-lavender: #F7F1FF;
  --text-cocoa:  #221F1D;
  --text-muted:  #BAB6C0;
}
body {
  background: var(--bg-milk);
  color: var(--text-cocoa);
}
Step 3 — Monokai palette
/* 3. Monokai syntax accents */
:root {
  --mk-pink:      #FC618D;
  --mk-purple:    #948AE3;
  --mk-green:     #7BD88F;
  --mk-cyan:      #5AD4E6;
  --mk-yellow:    #FCE566;
  --brand-tikka:  #BC7914;
}
절대 하지 말 것: Convex는 "amber on dark" 대시보드가 아니다. 실제는 크림/아이보리 warm backgrounds 위에 Monokai syntax palette가 얹힌 art/magazine 감성. 배경을 검은색으로 만들면 브랜드 인상이 정반대로 뒤집힌다. dark theme는 opt-in이고, 기본은 light.

09 — Rules

자주 틀리는 것들

DO
  • Warm cream backgrounds: #FFFDF4(milk) 기본, #F7F1FF(lavender) 보조.
  • Monokai syntax palette 5색을 브랜드 액센트로 활용: pink #FC618D, purple #948AE3, green #7BD88F, cyan #5AD4E6, yellow #FCE566.
  • 3-폰트 조합: GT America(sans) + Publico(serif, editorial) + VCR(pixel, retro label).
  • Dark text는 #221F1D(ebony) 또는 #2E1F1F(cocoa dark) — warm-tinted dark.
  • tikka masala #BC7914를 warm brand accent로 사용 (CTA, 강조).
  • Crayola crayon naming을 토큰에 보존: --color-tikkaMasala, --color-milk.
  • 10-step neutral ramp (n1..n10) 전체 등재.
  • Warm-tinted shadow: rgba(46,31,31,0.12) — cocoa 기반, 순수 black 아님.
  • Editorial/magazine 레이아웃: 큰 typography (5.5rem), pull quotes, drop caps.
DON'T
  • "amber on dark" 대시보드 스타일 — 실제는 cream light bg + Monokai 강조의 정반대.
  • 순수 흰색 #FFFFFF 페이지 bg — 실제는 warm off-white #FFFDF4.
  • Inter / Fira Code 사용 — 실제는 GT America / Publico / VCR.
  • Monokai palette 누락 — 이게 Convex의 실질 브랜드 컬러 (65회 최다 빈도).
  • color-brand / color-gray-* flat 네이밍 — Crayola descriptive 컨벤션 보존해야 브랜드 문화 유지.
  • 순수 black #000000 shadow — warm cocoa-tinted를 사용.
  • "generic developer dashboard" 레이아웃 — Convex는 art/editorial 감성.
  • dark theme 기본값 — Convex는 라이트 우선, 다크는 opt-in.
