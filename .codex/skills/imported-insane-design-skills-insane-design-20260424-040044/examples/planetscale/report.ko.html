Design System Report

PlanetScale — 시스템 폰트 + 블루 브랜드

Next.js + Tailwind CSS flat 네이밍. 커스텀 웹폰트 없이 시스템 sans 스택 그대로. 브랜드는 오렌지가 아닌 블루 #0B6EC5. 모든 서비스 중 가장 단순한 토큰 구조를 가진다.

Step 1 — 시스템 폰트
body {
  font-family:
    ui-sans-serif,
    system-ui,
    sans-serif,
    "Apple Color Emoji",
    "Segoe UI Emoji";
  font-weight: 400;
  /* @font-face 없음 */
}
code, pre {
  font-family:
    ui-monospace,
    SFMono-Regular, "SF Mono",
    Menlo, monospace;
}
Step 2 — Gray 팔레트
:root {
  --gray-50:  #fafafa; /* bg */
  --gray-100: #ebebeb;
  --gray-300: #c1c1c1;
  --gray-500: #818181; /* muted */
  --gray-700: #414141; /* text */
  --gray-900: #111111; /* dark */

  --text-primary: var(--gray-700);
  --bg-primary:   var(--gray-50);
}
body {
  background: var(--bg-primary);
  color: var(--text-primary);
}
Step 3 — 블루 브랜드
:root {
  --blue-600: #0B6EC5;
  /* canonical brand */
  --blue-500: #1E9DE7;
  /* link / focus ring */
  --blue-400: #4ab0f0;

  /* focus ring */
  .focus-ring:focus-visible {
    outline: 3px solid
      rgb(30 157 231 / 0.5);
    outline-offset: 2px;
  }
}
절대 하지 말 것: 오렌지 #F35815를 브랜드 색으로 쓰지 말 것. 2024년 이후 PlanetScale은 Vitess 엔터프라이즈 데이터플랫폼 톤으로 피벗하면서 UI는 블루 #0B6EC5 / #1E9DE7 중심이다. 오렌지는 legacy 로고 잔재에 가깝다.

Overview

테크 스택

자체 디자인 시스템 없이 Tailwind CSS 컬러 램프를 직접 참조하는 2계층 구조. 단순하지만 명확하다.

Framework

Next.js 마케팅 사이트 (Tailwind 컴파일 결과물)

CSS Architecture (2계층)

Color ramps: --{family}-{step} → raw hex
Semantic alias: --text-*, --bg-*, --border-* → ramp 참조

Naming

Tailwind 유틸리티: text-gray-700, bg-blue-600
가상 토큰명(color-brand 등) 없음. flat 네이밍만.

Font Loading

시스템 폰트 스택 (@font-face 없음)
macOS → SF Pro, Windows → Segoe UI로 자동 매핑

Foundations

타이포그래피 — weight 600이 주력

font-weight 히스토그램: 600(13회) > 500(6회) > 700(4회) > 400(4회). 제목과 강조에 600이 압도적. 폰트는 OS 시스템 sans (SF Pro / Segoe UI).

Token Size Weight Line-height
Heading Large heading-lg 32px 700 lh 1.2
Heading Medium heading-md 20 / 24px 600 lh 1.3
Heading Small heading-sm 18px 600 lh 1.4
Body — 일반 본문 텍스트 body 16px 400 lh 1.5
Small — 메타, 레이블 sm 14px 400 lh 1.5
Extra small — caption xs 12px 400 lh 1.45

CSS 스니펫

바로 복사해 사용할 수 있는 드롭인 스니펫이다. Tailwind config는 flat naming이므로 별도 설정 불필요.

CSS 스니펫 
/* PlanetScale — copy into your root stylesheet */
:root {
  /* Fonts — system stack */
  --font-sans: ui-sans-serif, system-ui, sans-serif,
               "Apple Color Emoji", "Segoe UI Emoji";
  --font-mono: ui-monospace, SFMono-Regular, "SF Mono", Menlo, monospace;

  /* Brand blue */
  --blue-600: #0B6EC5;
  --blue-500: #1E9DE7;
  --blue-400: #4ab0f0;

  /* Gray ramp */
  --gray-50:  #fafafa;
  --gray-100: #ebebeb;
  --gray-300: #c1c1c1;
  --gray-500: #818181;
  --gray-700: #414141;
  --gray-900: #111111;

  /* Semantic */
  --text-primary:   var(--gray-700);
  --text-secondary: var(--gray-550, #737373);
  --bg-primary:     var(--gray-50);
  --bg-inverted:    var(--gray-900);
  --border-primary: var(--gray-700);

  /* Danger / Success */
  --red-700: #c11027;
  --green-600: #22A652;

  /* Domain-specific */
  --text-postgres: #336791;
}

body {
  font-family: var(--font-sans);
  font-weight: 400;
  background: var(--bg-primary);
  color: var(--text-primary);
}

.focus-ring:focus-visible {
  outline: 3px solid rgb(30 157 231 / 0.5);
  outline-offset: 2px;
}
Rules

자주 틀리는 것들

오렌지를 브랜드로 착각하는 것이 가장 흔한 실수다. 단순한 구조이므로 과도한 추상화도 금지.

DO
  • Brand = #0B6EC5 (blue-600). 링크, CTA, 포커스 링 모두 블루 계열.
  • 시스템 sans 스택 유지 (ui-sans-serif, system-ui). Inter 강제 로드 불필요.
  • Tailwind 유틸리티 클래스 그대로 읽을 것 (bg-blue-600 text-white).
  • gray ramp 12단계 (gray-50 → gray-900)를 중립 축으로 사용.
  • 도메인 특화 색 보존: --text-postgres #336791 같이 DB 타입별 컬러.
DON'T
  • 오렌지 #F35815를 primary로 선언하지 말 것 (legacy/로고 색).
  • color-brand / color-ink 같은 가상 토큰명 사용 금지 — 실제 CSS는 --{family}-{step} flat.
  • Inter/IBM Plex Mono를 @font-face로 로드하지 말 것. 시스템 sans가 의도된 설계.
  • 단일 5색 팔레트로 환원 금지. 실제는 8 family × 10 step 램프 기반.
  • Dual-layer 카드 섀도(Stripe식) 적용 금지. Tailwind single ring/shadow가 기본.
