Design System Report

Warp -- --framer-*

https://warp.dev 가 실제로 서빙하는 CSS 에서 직접 뽑은 디자인 토큰, 타이포그래피, 컴포넌트 스펙. 모든 값은 CSS 커스텀 프로퍼티를 파싱한 결과이고, AI 추론이나 추측은 섞이지 않았다. 스크린샷: screenshots/light-hero.png

주의: 마케팅 사이트 vs 터미널 앱
이 문서는 warp.dev 마케팅 사이트만 다룬다. 터미널 앱 UI (다크 테마, cyan #01A4FF)와는 완전히 다른 디자인 표면이다. 마케팅 사이트는 따뜻한 크림 아이보리 (#FAF9F6) 배경의 라이트 테마이며, 채도 있는 컬러가 CSS 에 존재하지 않는 near-monochrome 디자인이다.
Quick Start 5분 안에 Warp처럼 만들기 -- 3가지만 하면 80%
1 . 폰트 body { font-family: "Inter", -apple-system, sans-serif; font-weight: 400; }
2 . 배경 + 텍스트 :root { --bg: #FAF9F6; --fg: #000000; } body { background: var(--bg); color: var(--fg); }
3 . CTA .cta { background: #000000; color: #FAF9F6; border-radius: 6px; font-weight: 500; }
절대 하지 말아야 할 것 하나: 배경을 #FFFFFF 순백이나 #0E0E10 다크로 두는 것. Warp 마케팅 사이트는 따뜻한 크림 아이보리 #FAF9F6이다. 터미널 앱과 마케팅 사이트를 혼동하면 전체 톤이 정반대가 된다.

03 . Typography

타이포그래피

폰트 스택, 굵기 정의, 타이포그래피 스케일. Matter Regular 는 유료 라이선스 (Displaay.co).

Font stack
Body font Inter (Google Fonts, 오픈소스) -- 59회 사용
Display font Matter Regular / Matter Medium (Displaay, 유료) -- 6회 사용
Code font Geist Mono (Vercel, 오픈소스) -- 6회 사용
Additional mono Fragment Mono (3), DM Mono (2), Matter Mono Regular (2)
Weight normal / bold 400 / 600
라이선스 주의: Matter Regular / Matter Medium 은 Displaay.co 의 유료 라이선스 폰트다. 대체하려면 Inter 를 그대로 쓰거나, DM Sans 나 Satoshi 가 비교적 가까운 대체재. Matter SQ (Square variant) 도 사이트에서 사용 중.

05 . Spacing

스페이싱

Framer 빌드라 spacing 토큰이 CSS 변수로 체계화되어 있지 않다. 대부분 인라인 스타일로 직접 지정된다.

관측 패턴: Framer 플랫폼 변수 6개만 존재. 실제 spacing 은 컴포넌트별 인라인 스타일로 설정된다. 관측된 주요 리듬: 섹션 80px~120px, 카드 24px~32px, 요소 16px~24px, 인라인 8px~12px.

08 . Layout Patterns

레이아웃 패턴

  • Layout: 중앙 정렬, 큰 헤드라인 상단 + 2-column 제품 카드
  • Background: cream #FAF9F6
  • H1: fluid clamp(56px, 9vw, 120px) / weight 400 / -0.02em

  • Max-width: ~1200px

  max-width: 1200px;

12 . Do / Don't

Do / Don't

실제 CSS 에서 검증된 규칙. Warp 마케팅 사이트 디자인을 재현할 때 흔히 틀리는 것들.

DO

  • 페이지 배경을 #FAF9F6 따뜻한 크림 아이보리로. 순백이 아니다.
  • 본문 텍스트를 순수 #000000 검정으로.
  • CTA 버튼: 검은 배경 #000 + 크림 텍스트 #FAF9F6, radius 6px.
  • 헤드라인을 Inter 400 또는 Matter Regular 400으로. 볼드(700)가 아님.
  • Mono 폰트는 Geist Mono.
  • 전체를 monochrome(흑백+회색)으로. 채도 있는 색은 스크린샷 안에서만.
  • 다크 섹션은 #121212. 순수 #000 아님.
  • radius 를 작게: 버튼 6px, 카드 8~10px.
  • Matter 라이선스 없으면 Inter로 통일.

DON'T

  • 배경을 다크(#0E0E10)로 두지 마라 -- 터미널 앱과 마케팅 사이트를 혼동하는 가장 흔한 실수.
  • cyan #01A4FF를 브랜드 악센트로 쓰지 마라 -- 마케팅 CSS 에 0회 등장.
  • 코드 폰트를 Hack으로 두지 마라 -- 실제 0회. Geist Mono가 정답.
  • 헤드라인을 font-weight: 700 이상으로 두지 마라.
  • 버튼에 그래디언트나 섀도우를 넣지 마라 -- flat black/white.
  • 카드에 두꺼운 보더나 큰 shadow 를 넣지 마라.
  • 채도 높은 컬러 팔레트를 만들지 마라 -- 22개 hex 중 채도 있는 색이 0개.
  • 순백 #FFFFFF를 페이지 배경으로 쓰지 마라 -- warm cream #FAF9F6과 다르다.
