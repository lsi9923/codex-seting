Design System Report

Dub.co — shadcn HSL + Satoshi + GeistMono

https://dub.co 의 CSS에서 직접 뽑은 디자인 토큰·타이포그래피·컴포넌트 스펙. Next.js + Tailwind + shadcn/ui 기반. 3-way 폰트 분리, 브랜드 퍼플 #855AFC, analytics chart 6색 팔레트.

Quick Start 5분 안에 Dub.co처럼 만들기 — 3가지만 하면 80%
1 · 폰트 3-way /* 1. 폰트 3-way 분리 */ :root { --font-satoshi: "Satoshi", "Satoshi Fallback", system-ui, sans-serif; --font-inter: "Inter", ui-sans-serif, system-ui, sans-serif; --font-geist-mono: "GeistMono", ui-monospace, Menlo, monospace; } body { font-family: var(--font-inter); } h1,h2,h3 { font-family: var(--font-satoshi); font-weight:700; }
2 · 라이트 + Tailwind neutral /* 2. 라이트 + Tailwind neutral */ :root { --bg: #FFFFFF; --bg-muted: #FAFAFA; --fg: #171717; --fg-alpha: rgba(0,0,0,0.67); /* 실제 빈도 2위 */ --border: #E5E5E5; } body { background: var(--bg); color: var(--fg); }
3 · 브랜드 퍼플 /* 3. 브랜드 퍼플 */ :root { --brand: #855AFC; /* #7B61FF 아님 — 실제 12회 */ } /* Analytics chart 6색 */ :root { --chart-1: #229DF3; --chart-2: #34A853; --chart-3: #EAB308; --chart-4: #F4950C; --chart-5: #F35066; --chart-6: #5CFF80; }
절대 하지 말아야 할 것: Dub의 brand 폰트는 Satoshi이며, Inter 단독이 아니다. headings와 stat 숫자에 Satoshi를 쓰지 않으면 "premium SaaS" 감성이 generic 대시보드로 변질된다. 그리고 정확한 brand 퍼플은 #7B61FF가 아니라 #855AFC다 (10 유닛 차이지만 다른 색).

03 · Typography

Satoshi(heading) + Inter(body) + GeistMono(stats)

3-way 폰트 분리가 핵심. stats 숫자와 short link는 반드시 GeistMono.

10 · Do / Don't

Do / Don't

지켜야 할 것과 피해야 할 것.

Do

  • 3-폰트 분리: Satoshi (display/heading) + Inter (body) + GeistMono (stats/short link/code).
  • 정확한 brand purple #855AFC — #7B61FF 아님.
  • shadcn/ui HSL channel token (hsl(var(--primary)/50%)) opacity modifier.
  • Tailwind neutral ramp 그대로 사용 (#FFFFFF → #171717).
  • Analytics chart 6색: #229DF3, #34A853, #EAB308, #F4950C, #F35066, #5CFF80.
  • Short link 표시는 반드시 GeistMono — link analytics 서비스 포인트.
  • Radius 5px(button) / 12-16px(card) / 9999px(pill).
  • Pink inset shadow inset 0 1px 0 0 rgba(243,80,102,.23)를 hover signature로.

Don't

  • #7B61FF brand purple 사용 — 실제는 #855AFC (10 유닛 차이).
  • Inter 단독 — Satoshi 누락 시 premium SaaS 감성 소실.
  • Stat 숫자를 Inter로 — 반드시 GeistMono.
  • color-brand / color-gray-* flat 네이밍 — shadcn HSL slot 사용.
  • #F9FAFB — 실제 0회. Tailwind #FAFAFA(neutral-50) 사용.
  • #6B7280 text-muted — 실제는 #737373 / #525252.
  • Heavy multi-layer shadow — Dub은 subtle card + pink inset hover.
  • 단일 brand color — analytics chart 6색 페어링 고려.
자주 틀리는 것들 기존 관행 vs 실제 값
쇼케이스 값 실제 값 이유
#7B61FF #855AFC CSS에서 12회 실제 등장. 10 유닛 차이지만 색조가 다름.
Inter 단독 Satoshi + Inter + GeistMono Satoshi 없으면 heading이 generic. GeistMono는 short link 전용.
#F9FAFB #FAFAFA Tailwind neutral-50. #F9FAFB는 Tailwind gray-50 — 실제 0회.
#6B7280 #737373 Tailwind neutral-500. gray-500과 neutral-500은 다른 값.
