Design System Report

Neon — 네온 그린 + 다크 캔버스

Next.js App Router + Tailwind v4 기반의 서버리스 Postgres 플랫폼. #2C6D4C halftone SVG 패턴이 3,602회 반복되는 decorative 배경이 Neon 시그너처다. CTA 텍스트는 흰색이 아닌 다크 #0C0D0D.

Step 1 — 폰트
body {
  font-family:
    var(--font-inter),
    "Inter", "Inter Fallback",
    ui-sans-serif, sans-serif;
  font-weight: 400;
}
code, pre {
  font-family:
    var(--font-geist-mono),
    GeistMono, ui-monospace,
    monospace;
}
Step 2 — 다크 캔버스
:root {
  --bg-canvas: #0C0D0D;
  --surface-1: #131415;
  --surface-2: #18191B;
  --surface-3: #242628;
  --surface-4: #303236;
  --border:    #494B50;
  --text:      #FFFFFF;
  --text-muted:#AFB1B6;
}
body {
  background: var(--bg-canvas);
  color: var(--text);
}
Step 3 — 브랜드 Neon Green
:root {
  --brand:       #00E599;
  --brand-hover: #34D59A;
  --brand-tint:  #E4F1EB;
  --decor-deep:  #2C6D4C;
  /* ↑ halftone SVG 배경 전용 */
}

.btn-primary {
  background: var(--brand);
  color: var(--bg-canvas);
  /* 텍스트는 흰색이 아닌 다크 */
  border-radius: 38px;
}
절대 하지 말 것: "No gradients" 원칙을 따르지 말 것. 실제 Neon 사이트는 #2C6D4C 짙은 네온 그린이 3,602회 반복되는 decorative SVG pattern/gradient로 hero와 섹션 배경을 채운다. halftone/gradient 텍스처는 Neon 브랜드 핵심 시그너처다.

Rules

자주 틀리는 것들

CTA 텍스트 색과 radius가 가장 흔한 실수다. "flat no gradients" 원칙은 이 사이트에 해당하지 않는다.

DO
  • 브랜드 네온 그린 #00E599 + hover #34D59A. 연한 tint #E4F1EB.
  • 다크 캔버스 #0C0D0D + surface 9-tier (#131415 → #494B50).
  • Mono 폰트는 GeistMono (Vercel OSS), var(--font-geist-mono).
  • Radius 38-40px pill 스타일. rounded-[38px].
  • CTA 텍스트는 #0C0D0D (다크) — 네온 그린 위에 검정 텍스트.
  • Decorative #2C6D4C halftone gradient를 hero 배경으로.
  • Shiki로 코드 블록 하이라이팅 (--shiki-color-text: #131415).
DON'T
  • JetBrains Mono 금지 → GeistMono.
  • "No gradients" 원칙 따르지 말 것 → 실제는 #2C6D4C halftone 3,602회.
  • Sharp 4-8px radius 금지 → soft 38-40px pill이 기본.
  • CTA 텍스트 색을 흰색으로 하지 말 것 → 네온 그린 배경 위에는 #0C0D0D.
  • #FF3621 (Databricks) / #0055FF customer logo hex를 브랜드로 착각 금지.
  • Pure black #000 canvas 금지 → #0C0D0D 사용.
  • Display weight 700을 본문에 적용 금지 → 본문은 500/600.
  • IBM Plex Sans를 주 폰트로 쓰지 말 것 → 레거시 잔존, Inter가 현재 주.
