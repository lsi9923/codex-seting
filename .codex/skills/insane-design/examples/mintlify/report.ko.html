Mintlify — Teal + Warm White 문서 플랫폼

Next.js + Tailwind v4 기반의 개발자 문서 플랫폼. Cool gray를 배제하고 warm palette로 구성했으며, Twoslash TypeScript hover popup이 시그너처 디테일이다. 브랜드 컬러는 teal #0C8C5E 단일이다.

Step 1 — 폰트
body {
  font-family:
    var(--font-inter),
    "Inter", "Inter Fallback",
    -apple-system, sans-serif;
  font-weight: 400;
}
code, pre {
  font-family:
    var(--font-geist-mono),
    "Geist Mono Fallback",
    ui-monospace, monospace;
}
Step 2 — Warm Light Palette
:root {
  --color-background-main: #FFFFFF;
  --color-background-soft: #F1EFED;
  /* warm off-white — cool gray 금지 */

  --color-text-main: #231F20;
  /* warm black — pure #000 금지 */

  --color-text-soft: #4A5565;
  --color-border-solid: #D1D5DC;
}
body {
  background: var(--color-background-main);
  color: var(--color-text-main);
}
Step 3 — 브랜드 Teal
:root {
  --color-brand:       #0C8C5E;
  --color-brand-light: #20808D;
  /* accent lime (dark CTA) */
  --color-accent-lime: #18E299;
}

/* Twoslash popup (시그너처) */
:root {
  --twoslash-popup-bg:    #F3F7F6;
  --twoslash-border-color:#DBDFDE;
  --twoslash-popup-shadow:
    rgba(0,0,0,.08) 0px 1px 4px;
}
절대 하지 말 것: 홈페이지 customer logo wall의 hex(PayPal #003087, Braintrust #FF5A00, Coinbase #4B73FF)를 Mintlify 브랜드 팔레트로 착각하지 말 것. 이 값들은 제3자 SVG 로고 색이며 Mintlify와 무관하다. 진짜 브랜드는 teal #0C8C5E 단일이다.

Rules

자주 틀리는 것들

customer logo wall 오염과 cool gray 사용이 가장 흔한 실수다.

DO
  • 브랜드 teal은 #0C8C5E (primary), #20808D (secondary). 밝은 lime #18E299는 accent.
  • Primary text는 warm black #231F20 — pure black 아님.
  • Surface는 warm off-white #F1EFED — cool gray 아님.
  • 폰트는 Inter (body) + Geist Mono (code). next/font 변수 방식.
  • Metric fallback (Inter Fallback, Geist Mono Fallback) 포함 필수.
  • 다크 모드는 invert 변수 패턴. --color-background-invert: var(--color-text-main).
  • Prose 토큰 (--tw-prose-pre-bg) 사용해 문서 본문 스타일링.
  • Twoslash popup (--twoslash-popup-bg)을 TS 코드 hover에 사용 — Mintlify 시그너처.
DON'T
  • Fira Code 쓰지 말 것 → 공식은 Geist Mono.
  • #0D9373 쓰지 말 것 → 실제 brand는 #0C8C5E.
  • pure black #000 text 쓰지 말 것 → warm #231F20.
  • cool gray surface (#F5F6F8) 쓰지 말 것 → warm #F1EFED.
  • Customer logo wall의 hex (#003087 PayPal, #FF5A00 Braintrust, #4B73FF Coinbase)를 Mintlify 브랜드로 착각하지 말 것.
  • 다크 모드를 두 세트 hex 나열로 구현하지 말 것 → invert 변수 flip 패턴.
  • 일반 Tailwind docs 템플릿 느낌 금지 → Twoslash / Prose 문서 특화 토큰 포함.
  • 본문 폰트를 px 절대값으로 고정 금지 → Prose em/% 사용.
