Age of Wonders: Planetfall -- --*

paradoxinteractive.com/games/age-of-wonders-planetfall/about 이 실제로 서빙하는 CSS 에서 직접 뽑은 디자인 토큰, 타이포그래피, 컴포넌트 스펙. 모든 값은 CSS 커스텀 프로퍼티를 파싱한 결과이고, AI 추론이나 추측은 섞이지 않았다.

Quick Start 5분 안에 Planetfall처럼 만들기 -- 3가지만 하면 80%
1 -- 폰트 body { font-family: "Source Sans Pro", sans-serif; font-weight: 400; color: #DED6C1; }

절대 하지 말아야 할 것 하나: 배경을 #000000 순수 검정으로 두는 것. Planetfall 의 다크 테마는 딥 네이비 블루 #1A1D48 과 SF-desert 톤 #122534 의 조합이다. 순수 블랙을 쓰면 게임 특유의 우주-사막 분위기가 사라진다.

08 -- Motion

모션

관측된 transition 패턴. 명시적 motion 토큰 변수는 없지만 일관된 패턴이 있다.

배경 전환 600ms ease 배경 이미지/색상 전환
버튼 색상 변경 200ms bg-color, border-color, color 동시 전환
불투명도 페이드 150ms ~ 500ms ease 페이드 인/아웃
Nav 스크롤 숨김 200ms transform translateY(-50px)
슬라이드 드로어 300ms ease-in-out 토너먼트 카드 슬라이드
일반 인터랙션 120ms ~ 450ms ease-out 전반적 UI 전환
09 -- Layout Patterns

레이아웃 패턴

히어로, 섹션 리듬, 반응형 브레이크포인트.

Hero

  • 레이아웃: 전체 폭 다크 배경 + 중앙 게임 로고
  • 배경: #1A1D48 (deep navy) + 게임 아트
  • H1: "Reclaim Your Future By Exploring The Past"
  • CTA: "Buy now" / "Watch trailer"

Section Rhythm

  • 패딩: 2rem 0
  • max-width: var(--screen-width-medium)
  • margin: auto (센터 정렬)
section {
  padding: 2rem 0;
  max-width: var(--screen-width-medium);
  margin-inline: auto;
}

Breakpoints

  • 480px: 2열 전환, 모바일 레이아웃
  • 640px: 스토어 링크 3열, 모바일 nav
  • 860px: 토너먼트 리스트 그리드
  • 1024px: 전체 그리드, nav 확장
  • 1320px: 스토어 링크 4열, 최대 너비

Navigation

  • position: fixed, top: 0
  • 배경: var(--theme-color-dark) = #1A1D48
  • 높이: 50px (모바일) / 72px (데스크톱)
  • 스크롤 숨김: translateY(-50px) + 200ms

13 -- Do / Don't

Do / Don't

실제 CSS 에서 검증된 규칙. Planetfall 디자인을 구현할 때 흔히 틀리는 것들.

DO

  • #1A1D48 을 게임 페이지 primary dark 배경으로 사용. 순수 검정이 아니라 딥 네이비.
  • #E5D195 골드를 모든 CTA 와 강조 요소에 일관되게 적용. Planetfall 시그너처.
  • 본문 텍스트는 #DED6C1 (sand). 순백이 아닌 따뜻한 아이보리.
  • heading 에 text-shadow: 0 0 24px #040507 가독성 halo 필수.
  • 버튼 radius 는 2px. SF/밀리터리 느낌의 핵심.
  • 버튼 letter-spacing 1px + uppercase 적용.
  • 호버 효과에 radial-gradient 골드 글로우 사용.
  • Source Sans Pro body + 게임 커스텀 heading 조합 유지.

DON'T

  • 배경을 #000000 순수 검정으로 두지 마라. 우주-사막 분위기가 사라진다.
  • 본문을 #FFFFFF 순백으로 두지 마라. 눈이 피로하고 게임 분위기가 깨진다.
  • 버튼 radius 를 8px 이상으로 올리지 마라. SF 톤이 무너진다.
  • --color-white 가 #FFFFFF 라고 가정하지 마라. 실제 값은 #EAE5DE.
  • 골드 악센트를 여러 색으로 분산하지 마라. #E5D195 하나로 통일.
  • heading 을 700 으로 두지 마라. 기본은 400.
  • text-shadow 를 제거하지 마라. 다크 배경 위 가독성의 핵심 장치.
  • 밝은 테마를 기본으로 가정하지 마라. 완전한 다크 테마 전용 서비스.
