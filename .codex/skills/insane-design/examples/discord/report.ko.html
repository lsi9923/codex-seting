Design System Report

Discord — Aurora Gradient System

https://discord.com 마케팅 사이트의 CSS에서 직접 뽑은 디자인 토큰·타이포그래피·컴포넌트 스펙. 4,181개 커스텀 프로퍼티, 10+ Aurora gradient 테마 — 모든 값은 CSS 직접 파싱, AI 추론 없음.

Quick Start 5분 안에 Discord처럼 만들기 — 3가지만 하면 80%
1 · 폰트 — 3-way 분리 /* 1. 폰트 — ABC Ginto triple */ :root { --font-display: "Abcgintodiscord", "ABC Ginto Normal", sans-serif; --font-nord: "Abcgintonord 800", "ABC Ginto Nord", sans-serif; --font-app: "gg sans", sans-serif; } body { font-family:var(--font-display); font-weight: 400; }
2 · 배경 — 순수 black /* 2. 배경 + Blurple */ :root { --bg: #000000; /* 1145회 — 실제 dominant */ --bg-elev: #23272A; /* 앱 내부 surface */ --text: #FFFFFF; --blurple: #5865F2; /* 브랜드 앵커 */ } body { background: var(--bg); color: var(--text); }
3 · Aurora gradient 시그니처 /* 3. Aurora signature */ .hero { background: linear-gradient( 135deg, #5865F2 0%, #B377F3 50%, #FF6AEF 100% ); } /* 10+ 테마: aurora, cotton-candy, forest, crimson-moon, hanami... */
절대 하지 말아야 할 것: Discord 마케팅 사이트의 정체성은 #5865F2 Blurple 단색이 아니라 10+ 테마의 Aurora gradient 시스템이다. aurora, chroma-glow, cotton-candy, crimson-moon, forest, hanami, lofi-vibes 등 수십 개 gradient가 각 섹션 배경/카드에 배치된다. Blurple 단색으로만 만들면 "generic 게이머 다크 채팅"이 된다.

03 · Typography

마케팅은 Nord 800, 앱은 gg sans

3가지 폰트를 역할별로 분리. 16px(374회)와 20px(356회)가 body 양대 축. 127 unique sizes — 섹션마다 다른 hero 스케일.

05 · Spacing

스페이싱(간격 체계) — raw + semantic 2-tier

2-tier 구조: --size-{0..192} raw px 원자 계층 → --size-{xxs..xxl} semantic alias. 컴포넌트는 semantic alias 우선 사용.

10 · Do / Don't

Do / Don't

지켜야 할 것과 피해야 할 것.

Do

  • Blurple #5865F2를 primary CTA로. 2020 리브랜딩 후 current 브랜드.
  • 순수 black #000000 페이지 bg — 실제 1145회 dominant. Aurora gradient가 black 위에서 최고 vivid.
  • Aurora gradient 시스템 활용: 10+ 테마를 섹션마다 교체. aurora, cotton-candy, forest, crimson-moon 등.
  • 3-폰트 분리: ABC Ginto Normal (display), ABC Ginto Nord 800 (heavy impact), gg sans (app UI).
  • 2-tier 스페이싱: raw --size-{N} + semantic --size-{xxs..xxl} alias.
  • Neon accent 4종: mint, lavender, pink, baby-blue.
  • component-slot 네이밍: --shadow-low, --shadow-ledge, --shadow-button-overlay.
  • Radius는 컴포넌트별 다르게: 8px(button), 16px/40px/88px(card).

Don't

  • Blurple 단색만 — Discord 마케팅 정체성은 multi-accent + gradient.
  • Legacy Blurple #7289DA — 2020년 이전 색. current 브랜드 아님.
  • gg sans를 마케팅 heading에 — 앱 UI 전용. 마케팅은 ABC Ginto Nord.
  • #313338 / #2B2D31 — 앱 내부 surface. 마케팅 페이지에는 0회.
  • Flat 카드 — Aurora gradient 없으면 marketing 감성 소실.
  • 단일 radius 값 — 컴포넌트별 8px ~ 88px 다양하게.
  • "pure black 쓰지 마라" 규칙 적용 — 실제 반대. #000000이 1145회 dominant.
  • color-brand / color-gray-* 네이밍 — Discord는 --size-*, --space-*, --bg-gradient-* tier.
