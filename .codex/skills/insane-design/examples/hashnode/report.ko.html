Hashnode — Tailwind v4

Suisse International + Tailwind v4 @theme + 얇은 브랜드 블루 덮기. 자체 토큰 시스템 없음 — 이게 설계 철학.

STEP 1 — Font (next/font/local)
body {
  font-family:
    var(--font-suisse-intl),
    system-ui, -apple-system,
    BlinkMacSystemFont,
    "Segoe UI", sans-serif;
  font-weight: 400;
}
code, pre {
  font-family:
    var(--font-suisse-mono),
    ui-monospace, "SF Mono",
    monospace;
}
STEP 2 — Neutral Canvas
:root {
  --bg: #ffffff;
  --fg: #101828;  /* slate-900 */
  --fg-muted: #4a5565;
  --border: #e5e7eb;
}
body {
  background: var(--bg);
  color: var(--fg);
}
STEP 3 — Brand Blue
:root {
  --brand: #1D52DE;
  --brand-soft: #F5F9FF;
}
a { color: var(--brand); }

/* CTA button */
.btn-primary {
  background: #1D52DE;
  color: #ffffff;
  border-radius: 8px;
  font-weight: 600;
}

단일 base 4px — p-{n} = n×4px

Tailwind v4 convention: --spacing: 0.25rem (4px) 하나만 정의. 나머지는 calc(var(--spacing) * n) 자동 계산.

자주 틀리는 것들

Inter/Fira Code 대체 폰트 오류, 허구 neutral 값, 자체 토큰 시스템 생성 — 반복 패턴

DO
  • Suisse International + Suisse Mono 사용. next/font/local로 로드, --font-suisse-intl CSS 변수로 노출.
  • Tailwind v4 @theme이 기본. --color-*-{N}이 전부 :root에 주입.
  • 브랜드 블루는 #1D52DE + #F5F9FF soft bg.
  • Neutral은 Tailwind slate/gray 팔레트 혼합. #101828 / #4a5565 / #e5e7eb.
  • 블로그 본문에 Prose 플러그인 사용. em/% 단위 타이포.
  • Hairline은 #0000001a (α10). 21회 실사용.
  • Weight 600 semibold를 강조 기본값으로. 700/400과 혼합.
DON'T
  • Inter를 폰트로 쓰지 말 것 → 실제 CSS에 Inter 0회 등장.
  • Fira Code를 코드 폰트로 쓰지 말 것 → 실제는 Suisse Mono.
  • #2962FF를 브랜드 블루로 쓰지 말 것 → 실제는 #1D52DE.
  • #1C1C1C / #F5F5F5 같은 허구 neutral 쓰지 말 것 → 실제는 Tailwind slate/gray.
  • 본문 폰트를 18px 절대값으로 고정하지 말 것 → Prose em/% 컨벡션 사용.
  • 자체 브랜드 토큰 시스템을 만들지 말 것 → 없다. 철학적으로 없음.
  • BEM semantic 클래스(button-primary-bg)를 찾지 말 것 → 없다. 유틸리티만.
